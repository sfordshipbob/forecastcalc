const quarterEndDates = [ new Date(currentDate.getFullYear(), 2, 31), // Q1 new Date(currentDate.getFullYear(), 5, 30), // Q2 new Date(currentDate.getFullYear(), 8, 30), // Q3 new Date(currentDate.getFullYear(), 11, 31) // Q4 ]; document.getElementById('mostLikelyValue').textContent = mostLikely.toFixed(2); document.getElementById('bestCaseValue').textContent = bestCase.toFixed(2); }); --- # Salesforce Input Calculator This tool is designed to help you calculate Salesforce inputs based on your close date, number of shipments per month, and revenue per shipment. The tool provides you with the necessary values to manually enter into Salesforce. ## How to Use 1. **Enter the Close Date**: Select the date when the deal is expected to close. 2. **Enter the Number of Shipments per Month**: Input the number of shipments you expect per month. 3. **Enter the Revenue per Shipment**: Input the revenue expected per shipment. 4. **Calculate**: Click the "Calculate" button to see the results. The tool will display the following values: - SPM: Stays constant at 21000. - Orders per Day: Calculated based on shipments per month. - Revenue per Order: Same as the revenue per shipment. - Days Left in Quarter: Calculated based on the close date. - Commit Forecast: Enter this value manually. - Most Likely: Calculated as Best Case divided by 1.35. - Best Case: Calculated based on Orders per Day, Revenue per Order, and Days Left in Quarter. You can then manually enter these values into Salesforce. ## Example Here's an example of what the tool looks like: ![Example Image](example.png) ## Technology Stack - HTML - CSS - JavaScript ## Hosting This tool is hosted on GitHub Pages. ## License This project is licensed under the MIT License - see the [LICENSE.md](LICENSE.md) file for details. --- MIT License Copyright (c) [Year] [Your Name] Permission is hereby granted, free of charge, to any person obtaining a copy of this software and associated documentation files (the "Software"), to deal in the Software without restriction, including without limitation the rights to use, copy, modify, merge, publish, distribute, sublicense, and/or sell copies of the Software, and to permit persons to whom the Software is furnished to do so, subject to the following conditions: The above copyright notice and this permission notice shall be included in all copies or substantial portions of the Software.
